Workshop Website

Developed a workshop website that enables organizations/individuals to create event sessions and allows participation from other users.

Designed the website with three functional roles: Teacher, Student, and Admin.

The Teacher role allows the creation of papers/subjects, and the setup of sessions with the ability to set up the subject, teacher name, and session location.

The Student role allows the viewing of sessions created by the teacher, joining a session, and viewing all the sessions they are currently participating in.

The Admin role has all the privileges of the Teacher role, the ability to terminate specific users, and the ability to change user roles.

Programming Languages Used:

HTML and CSS (Front End)

Python (Back End)

MYSQL (Database)
