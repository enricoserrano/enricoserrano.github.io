Who wants to be a millionaire Java App

This Java app is inspired by the popular game 'Who wants to be a millionaire.'

It presents the player with 15-set randomised questions in which they get a reward for each question answered correctly

At the end of every question answered correctly, the game will ask the player if they wish to continue or cash out of the game

The player is given one lifeline (50/50), which will remove the other two choices leaving the player to choose between the remaining ones

This game features a leaderboard which will display the top 3 player's results

Programming Languages Used:

Java and Java Swing (Backend and Frontend

Apache Derby (Leaderboard, Questions, Choices, Answers)
