Near-miss Application

This is a Research and Development project made for our client WorkSafe

It aims to increase the safety of workers in the work field by allowing workers to:

Take a photo of a Near-miss or incident in the workplace

Briefly describe the incident, location and time

Uses automatic geolocation tagging to determine the current location of the worker

Uses the 'Speech to text' feature which allows workers to describe the Near-miss without having to type in the fields

Fully responsive web-application to cater for both mobile and computer systems

Has an admin feature to manage, track, and resolved near-miss cases submitted by workers

Programming Languages Used:

HTML & CSS (Front-End)

Javascript

PHP (Back-End)

MYSQL (Database)
