Near-miss Application

Developed a Research and Development project for client WorkSafe, aimed at increasing workplace safety.

Implemented features that allow workers to take a photo of a near-miss or incident, provide a brief description, and use automatic geolocation tagging to determine the current location of the worker.

Incorporated speech-to-text functionality for ease of use, allowing workers to describe near-misses without typing.

Designed as a fully responsive web-application that is compatible with both mobile and computer systems.

Included an admin feature that allows management to track, manage and resolve near-miss cases submitted by workers.
