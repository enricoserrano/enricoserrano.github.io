De-Crypt It Unity Game

De-Crypt It is a unity game made as a group project for our Software Development Practice (SDP) at AUT

It is a first-person game which allows the player to interact with objects, find hints, and complete puzzles, to escape

It has three game modes (easy, medium, and hard) difficulty to support a wide range of players with their escape room experience

Easy Game Mode

15 Minutes of Gameplay

Medium Game Mode

30 Minutes of Gameplay

Hard Game Mode

60 Minutes of Gameplay

It has an online leaderboard which tracks player's results and displays them on the leaderboard (top 5)

This project also features the use of AGILE and SCRUM to maintain the project schedule, developmental process, and overall collaboration as a group

Programming Languages Used:

Primarily C# and Unity tool

Dreamlo (plugin for the database)
