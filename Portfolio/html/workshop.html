Workshop Website

This workshop website allows an organisation/individual to create an event session which allows other people to participate.

It consists of 3 functioning roles which are Teacher, Student, and Admin

Teacher Role can:

Create a Paper/Subject

Create a session

Upon creating a session, they will be able to setup subject, teacher name, and session Location

Student Role can:

View the sessions created by the teacher

Have the ability to join a session

View all the sessions they are currently participating in

Admin Role can:

Have all the privileges of a teacher role

Be able to terminate specific user

Have the ability to change the user role

Programming Languages Used:

HTML and CSS (Front End)

Python (Back End)

MYSQL (Database)
